Key Design Decisions

Single Codebase

MCP server + admin dashboard = one Next.js app. Shared DB, shared auth, shared types. Deploy once.

Project = Vercel Project

Each published app is its own Vercel project. Isolated deployments, independent URLs, separate logs.

Tenant = Supabase Row

Multi-tenancy via Row-Level Security. Each company is a tenant. All data isolated at the DB level. Ready for SaaS from day one.

Secrets Never in Code

API keys stored encrypted in Supabase. MCP server injects them at deploy time as Vercel env vars. Claude never sees raw secrets.
