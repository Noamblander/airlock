External Services & API Keys Needed

Vercel

  • Team account (Pro plan)
  • API token for deploy
  • OAuth app for tenant onboarding

Supabase

  • Project for platform DB
  • Auth configured with SSO
  • Service role key (server only)

Anthropic

  • MCP OAuth registration (for Claude.ai remote MCP)
  • No API key needed — employees use their own Claude

MVP Scope Summary

In MVP

  • MCP server with deploy, secrets, database, projects, usage tools
  • Admin dashboard (gallery + usage + secrets + users)
  • SSO auth (Google Workspace first)
  • Auto auth middleware injection
  • CLAUDE.md generation
  • Folder organization
  • Hebrew / RTL support
  • Deploy history + rollback
  • Basic agent scheduling (cron)

Post-MVP

  • Azure AD / SAML auth
  • Webhook-triggered agents
  • Custom domains for published apps
  • Billing / plan tiers for SaaS
  • Team permissions (view/edit per folder)
  • App templates gallery
  • Real-time collaboration (Supabase Realtime)
  • CLI tool (npx org-ai setup)
